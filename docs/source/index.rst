
This site contains the documentation for Compute_3d!
====================================================

Compute_3d is a Python package for computing 3D coordinates from 2D images,
from two dimensional data sets given by user input, data files or folders.
Some basic examples can be found in :ref:`Examples <examples>` and a
quickstart guide in :ref:`Quickstart <quickstart>`. A full API reference is
available in :ref:`API <api>`.
