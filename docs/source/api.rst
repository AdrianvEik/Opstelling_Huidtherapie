API
###

This simple package contains code to compute points in between data sets using
different interpolation schemes.

Constructor class
=================

The constructor class is the class that is used to create the object containing
the extra data points that are interpolated from the original data. The mathematical
justification and proof is given in the :ref:`theory <interpolation_theory>`.
